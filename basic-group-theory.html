Basic Group Theory

This chapter develops the foundational notions of group theory. We assume familiarity with the basic properties of the set of complex numbers $\mathbb C$, and its subsets $\mathbb N$, $\mathbb Z$, $\mathbb Q$ and $\mathbb R$.

Section 1

Our central motivation from this chapter arises from the set of all functions whose domain and codomain is a given set $S$. Suppose that $S$ is any nonempty set and $f,g:S\to S$ are functions. Then it is well known that the composition $f\circ g$ is also a function with domain $S$ and codomain $S$. This is an example of a binary operation on a set.

Definition 1.1. A binary operation on a nonempty set $G$ is a function with domain $G\times G$ and codomain $G$.

If $S$ is any nonempty set and $M(S)$ is the set of all functions $f:S\to S$ then function composition is a binary operation on $M(S)$. Other examples of binary operations are addition and multiplication on $\mathbb N$, $\mathbb Z$, $\mathbb Q$, $\mathbb R$, and $\mathbb C$. Subtraction is a binary operation on $\mathbb Z$, $\mathbb Q$, $\mathbb R$, and $\mathbb C$.
